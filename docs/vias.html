Vías del Barrio Potosí

Este mapa interactivo muestra la malla vial del barrio Potosí, localizado en la localidad de Suba, Bogotá.

Desarrollado con Leaflet, la biblioteca JavaScript líder para mapas interactivos móviles y de escritorio, este proyecto combina simplicidad, rendimiento y usabilidad.

Leaflet, con solo 42 KB, incluye todas las funciones esenciales: capas de teselas, marcadores, popups, GeoJSON y soporte para dispositivos móviles con arrastre, pinch-zoom y animaciones suaves.

Así se crea un marcador con popup en Leaflet. En este proyecto, se usan múltiples marcadores para representar las intersecciones de las vías dentro del barrio.
